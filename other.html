I last updated this site on Sunday 15th of May, 2022.

This website is run off of a small Scaleway VPS Linux server, utilising Debian 9 Stretch as the operating system.
The webserver is a fork of thttpd, a great static/CGI http server, called sthttpd; which adds a few security patches.
SSL/HTTPS is handled through stunnel using a Let's Encrypt certificate. Basically, this simply redirects the HTTPS queries to the static HTTP server, meaning that the HTTP server doesn't need to deal with the specifics of SSL and the SSL layer doesn't need to know anything about HTTP. Simple and secure, hopefully.

The website itself is all just static hand-coded HTML and CSS. It's not the most efficient nor pretty; but it's simple, fast, portable, secure, and I think does the job fine.
